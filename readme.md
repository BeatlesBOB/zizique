
# Zizique

Application utilisant expo et react-native ainsi que socket.io afin de crée un blindtest multijoueur basé sur l'api spotify


## Installation

Install zizique with npm

```bash
  git clone https://github.com/BeatlesBOB/zizique
  npm install
  cd zizique
  expo start
```
